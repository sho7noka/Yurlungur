# PyDcc
MEL and MaxScript Python wrapper andmore.

* better Wrapper
* Qt helper
* CUI Util

### Semantics
- Object

    ```lua
    -- MXS
    $.key[i].value = ""
    ```
    
    ```mel
    // MEL    
    setProperty "" ""
    ```
    
    ```python
    # python
    p.value()    
    ```
